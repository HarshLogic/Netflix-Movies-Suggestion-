# 🎬 Netflix Movies & TV Shows Clustering

## 📌 Project Overview

This project focuses on analyzing and clustering Netflix Movies and TV Shows using Machine Learning techniques. The goal is to group similar content based on features like genre, description, and other metadata to uncover hidden patterns and improve content recommendations.

---

## 🚀 Key Features

✨ **Content Clustering using ML**

* Applied clustering algorithms to group similar movies and TV shows

📊 **Exploratory Data Analysis (EDA)**

* Performed in-depth analysis to understand trends in Netflix content
* Visualized patterns like genre distribution, release trends, etc.

🧹 **Data Cleaning & Preprocessing**

* Handled missing values and inconsistent data
* Processed text data (descriptions, genres) for modeling

🧠 **Text Vectorization**

* Converted textual data into numerical form using techniques like TF-IDF

📌 **Unsupervised Learning**

* Implemented clustering algorithms (e.g., K-Means)
* Identified meaningful clusters of content

📈 **Visualization of Clusters**

* Used plots to represent how content is grouped
